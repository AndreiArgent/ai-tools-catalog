 # Личный каталог AI-инструментов

## 1. LLM (Large Language Models)
- [Llama](https://github.com/meta-llama/llama) — семейство больших языковых моделей от Meta. Этот репозиторий содержит код для запуска и примеры использования (Llama 2). ⭐ 59.5k

## 2. VLM (Vision-Language Models)
 - [LLaVA](https://github.com/haotian-liu/LLaVA) — одна из первых и самых популярных моделей, которая понимает и картинки, и текст. ⭐ 24.9k
 - [Qwen-VL](https://github.com/QwenLM/Qwen-VL) — мультимодальная модель от Alibaba Cloud... ⭐ 6.7k

## 3. Мультимодальные модели
- [ImageBind](https://github.com/facebookresearch/ImageBind) — ImageBind One Embedding Space to Bind Them All ⭐ 9k
 

## 4. SLM (Small Language Models)
 - [Phi-3-mini](https://github.com/marketplace/models/azureml/Phi-3-mini-4k-instruct) — компактная (3.8B) модель от Microsoft. Отлично работает на ноутбуках для рассуждений, кода и математики. Поддерживает контекст 128K токенов [citation:5][citation:7].

## 5. Reasoning Models
- [DeepSeek-R1](https://github.com/deepseek-ai/DeepSeek-R1) — модель с открытыми весами, специализирующаяся на цепочках рассуждений (Chain-of-Thought). Решает сложные математические, логические и программные задачи, конкурирует с OpenAI o1. ⭐ 92k
